所以就是数数字中o的数量，然后我写了个js函数

```javascript
function calcOCounts(number){
	const map = [1, 0, 0, 0, 0, 0, 1, 0, 2, 1]
	let str = [...String(number)]
	let count = 0
	for(let i=0 ; i<str.length; i++){
		count += map[str[i]]
	}
	console.log(count)
}
calcOCounts(1111)
calcOCounts(1289)
calcOCounts(2256)
calcOCounts(3388)
calcOCounts(9090)
calcOCounts(1868)
```
